### water-talker

IM 聊天示例

#### 环境

1、准备 `HTTP` 服务器

2、安装 `Node4.0+` 

3、运行环境 `Chrome45+`、`Safari10+` (演示项目， IE 等兼容性较差的浏览器未做适配)

#### 预览

1、将项目放置 HTTP 服务器下

2、项目根目录执行 `npm install`

3、启动服务 `npm run server` 

4、访问 IM `http://*****/water-talker/client/index.html?userId` userId 请替换成当前用户 Id
